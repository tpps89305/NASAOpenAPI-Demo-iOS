# NASA Open API Demo
This is an iOS APP Demo which show [Astronomy Picture of the Day (APOD)](https://api.nasa.gov/).

# Technical Specifications
- Language: Swift 5
- IDE: AppCode 2021.1 with Xcode 12.4
- No third party package(plugin)
- MVVM architecture
